Мета роботи:

  • Поглибити та закріпити знання з React
  • Придбати практичні навички з створення веб додатків за допомогою React.

Порядок виконання роботи

  1. Повторити основні конструкції і поняття, розглянуті на лекції.
  2. Створити програму першого завдання.
  3. Перевірити правильність роботи програми, згідно завдання.
  4. Текст програми та результат роботи занести у звіт.
  5. Повторити пункти 2-4 для решти завдань.
  6. Перевірити правильність роботи програми.

Теоретичні відомості

  • Лекція з теми
  • https://uk.reactjs.org/docs/hello-world.html

Індивідуальне завдання.

Завдання 1. Виведення тексту

Вивести текст "React Native is cool!" в центрі екрану, як показано на картинці

 



Завдання 2.1 Вивід елементів масиву users на екран

Вивести всі імена з масиву users на екран

Завдання 2.2 Композиція компонентів

Створити окремий компонент для відображення елементу списку. Компонент може бути оформлений як за допомогою класу так і функції, на Ваш розсуд. Допомога: https://uk.reactjs.org/docs/components-and-props.html

Завдання 3. Показати/сховати елемент на екрані

Необхідно при натисканні на кнопку показати/сховати текст на екрані. Також змінити напис на кнопці з "Hide Element Below" на "Show Element Below". Допомога: https://uk.reactjs.org/docs/conditional-rendering.html

Завдання 4. Введення даних

Необхідно відобразити введені дані, що вводяться.

Допомога: https://reactnative.dev/docs/textinput#onsubmitediting

Завдання 5. Доступність кнопки

Зробити кнопку недоступною, якщо в поле не введено жодного символу

Завдання 6. Сума двох чисел

Обчислити та вивести суму двох чисел
