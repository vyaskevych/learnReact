Додаток «Список справ»

Необхідно переробити додаток список справ в додаток з використанням функціональних компонентів та хуків.  

Завдання 1. Додаток «Список справ»

В якості заготовки додатку пропонується взяти за основу приклад додатку з посилання

Створіть новий додаток, вставьте в файл app.js код з файлу 1-todoApp.js. Запустить на виконання та переконайтесь, що додаток працює.

Додаток має суттєвий недолік, неможливо задати опис нової справи, виправте це. Додайте компонент TextInput. Налаштуйте його, щоб він працював. Текст, що введений в поле TextInput при натисканні на Button додається в список справ, при цьому поле TextInput очищається. Також потрібно передбачити, неможливість додавання порожньої справи. 

Завдання 2. Перша зміна додатку

Наступні кроки по суті не змінюють принцип роботи додатку, але змінюють його структуру (за прикладом лекції)

До виконання завдання ознайомтесь з використанням хуку стану (https://uk.reactjs.org/docs/hooks-state.html)

Компонент Todo виносимо в окремий файл в папку components. В файлі App.js підключаємо цей компонент за допомогою імпорту. Перевіряємо працездатність додатку. 

Далі основний компонент App переписуємо з класового вигляду у функціональний.  Імпортуємо hook useState з бібліотеки React. Створюємо за допомогою useState зміну (н-д:  text ) для збереження стану тексту з поля введення та функцію для зміни цього стану (н-д: setText) . Аналогічно змінюємо state - todos та виклик функції setState(). На нову функцію та зміну, що повертаються під час виклику useState.

Перевіряємо працездатність додатку. 

Завдання 3. Підключення useReducer

До виконання завдання:

  • ознайомтесь з використанням хуку useReducer(https://uk.reactjs.org/docs/hooks-reference.html#usereducer)
  • збережіть робочий проект, як новий з іншим ім'ям

Створіть папку reducers та в ній файл todos.js. В ньому створіть const initialState. Де для первірки задайте масив todos з прикладом справи, з всіма полями.

Додайте заготовку функції reducer, яка повинна приймати state та action. Не забувайте про експорт створених елементів (initialState, reducer). В основному файлі App імпортуйте initialState та reducer з файлу todos.js. Також імпортуйте hook useReducer з бібліотеки React. Викличте useReducer з створеними  initialState та reducer. В отриманому після виклику функції useReducer,  state містіться todos, який можна вивести. Запустіть додаток, та переконайтеся, що дані з initialState відображаються. 

Завдання 4. Реалізація функції reducer

В файл todos.js додаємо для зручності роботи const types, яка буде описувати типи дій для обробки даних, наприклад:

const types = {
    ADD: 'ADD',
    REMOVE: 'REMOVE',
    TOGGLE: 'TOGGLE'
}

Далі в тому ж файлі todos.js реалізуємо функцію reducer. Для цього переносимо логіку роботи з даними з файлу, що описує компонент (App.js) у функцію reducer файлу todos.js, наприклад:

export function reducer(state, action) {
    switch (action.type) {
        case types.ADD: return { ...state, todos: [...state.todos, { id: Date.now(), text: action.payload, checked: false }] }
        …
    }
}

Змінюємо функцію addTodo. Викликаємо в ній функцію dispatch, та передаємо їй об'єкт, що містить відповідні type та payload, наприклад:

const addTodo = () => {
    dispatch({ type: 'ADD', payload: text });
    …
  }

Запускаємо додаток, та переконуємося, що нові справи додаються.

Аналогічно реалізуємо функції removeTodo та toggleTodo. Перевіряємо працездатність додатку. 

! Опишіть у звіті сутність reducer та код функції reducer.  

Завдання 5. Створення actionsCreators

В файл todos.js створюємо об'єкт actionsCreators. Полями об'єкти є  функції, які повертають об'єкти з відповідними type та payload. Це дасть можливість в основному компоненті не створювати відповідні об'єкти а лише викликати функції, які і будуть створювати необхідні об'єкти. Приклад actionsCreators: 

export const actionsCreators = {
    add: (text) => ({ type: types.ADD, payload: text }),
    …
}
В файл App.js імпортуємо actionsCreators. Та в dispatch передаємо виклик відповідної функції з actionsCreators, наприклад: 

const addTodo = () => {
    dispatch(actionsCreators.add(text));
    …
  }
Так само робимо для всіх викликів dispatch.
